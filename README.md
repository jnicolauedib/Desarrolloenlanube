
# Pegasus Global Cars

**Pegasus Global Cars** es una plataforma digital para la **compra y venta de vehículos de lujo importados**. Conecta compradores interesados en coches de alta gama con vendedores o concesionarios, haciendo el proceso más simple y seguro.

## Funcionalidades

- Ver información detallada de cada vehículo.
- Fotos y especificaciones de alta calidad.
- Historial y precios de los coches.
- Comunicación directa entre compradores y vendedores.
- Transacciones seguras y confiables.

## Cómo funciona

1. Regístrate como comprador o vendedor.
2. Explora el catálogo de vehículos de lujo.
3. Revisa detalles e historial de cada coche.
4. Contacta al vendedor directamente.
5. Compra de manera segura y profesional.
